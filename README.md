# Sunflaser Laser Rangefinder Modules – Technical Documentation

**Eye-safe, high-shock tolerant 1535nm miniature laser rangefinder modules for optoelectronic system integration.**

## Overview

Sunflaser offers a family of compact, ruggedized laser rangefinder modules based on 1535nm erbium-glass technology. These modules are designed for easy integration into handheld devices, vehicle-mounted systems, and UAV payloads, delivering reliable distance measurement in harsh environments.

**Key features:**
- 1535nm wavelength – eye-safe at all distances
- High shock resistance (>1000g)
- Compact and lightweight design
- Low power consumption
- Multiple output interfaces (UART, etc.)

## Applications

- **Handheld optoelectronic devices** – rangefinders, observation scopes
- **Vehicle optoelectronic systems** – driver assistance, fire control
- **UAV/robot payloads** – reconnaissance, targeting, obstacle avoidance
